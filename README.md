# EveryFit Business UI

Рабочий frontend-прототип по макетам из Figma. Сейчас данные хранятся в состоянии приложения; сетевые запросы намеренно не подключены.

## Запуск

```bash
pnpm install
pnpm dev
```

Production-сборка:

```bash
pnpm build
```

## Реализовано

- список челленджей с поиском и фильтрацией;
- создание целевого и соревновательного челленджа;
- командный формат участия;
- живой мобильный предпросмотр;
- локальное сохранение черновика и публикация;
- загрузка логотипа и обложки на уровне UI;
- адаптивный layout и мобильная навигация.

В интерфейс включён только продуктовый сценарий челленджей. Отдельный макет занятий не используется.

## Подготовка к API

Доменные типы находятся в `src/types.ts`, тестовые данные — в `src/data.ts`. На этапе интеграции локальные операции в `App.tsx` нужно заменить методами API-клиента, не меняя компоненты формы и таблицы.
